Contents

This repository links to several binary distributions related to the MONDO EU project. Please note that the links below are not meant to be browsed, but rather copied and pasted into your Eclipse instance or your Maven POM file.

  • Eclipse update site with remote clients for the MONDO server.
  • Maven repository produced from the Eclipse update site using eclipse:to-maven.
  • Eclipse update site with a fixed version of the ECF HttpClient4 file transfer provider that enables SNI support for remote p2 repositories.
  • Ecore2Thrift update site: generates .thrift files from annotated Ecore metamodels.
